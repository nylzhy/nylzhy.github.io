Markdown Syntax Guide

Markdown 基础语法及格式的样本文章，快速入门Markdown

2019-03-11
阅读时长: 2 分钟

本文提供了一个可以在 Hugo 内容文件中使用的基本 Markdown 语法示例，还展示了基本 HTML 元素是否在 Hugo 主题中使用 CSS 进行装饰。

标题

下面的 HTML <h1>—<h6> 元素代表了六个级别的章节标题。 <h1> 是最高的部分级别，而 <h6> 是最低的。

段落

这个世界，总有你不喜欢的人，也总有人不喜欢你。这都很正常。而且，无论你有多好，也无论对方有多好，都苛求彼此不得。因为，好不好是一回事，喜欢不喜欢是另一回事。

刻意去讨人喜欢，折损的，只能是自我的尊严。不要用无数次的折腰，去换得一个漠然的低眉。纡尊降贵换来的，只会是对方愈发地居高临下和颐指气使。没有平视，就永无对等。

来自 莫言 大佬

引用

blockquote 元素表示从另一个来源引用的内容，可选地带有必须在footer或cite元素内的引用，以及可选的内联更改，例如注释和缩写。

不引起注意的引用

Tiam, ad mint andaepu dandae nostion secatur sequo quae. Note that you can use Markdown syntax within a blockquote.
